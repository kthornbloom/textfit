textfit.js
Fit text within its container
  • Include<script src="js/textfit.js"></script> after calling jQuery.
  • Apply a class of textfit to any element containing a single line of text.
  • Set data-textfit-min="10px" on the element to set an optional minimum size.
  • Set data-textfit-max="60px" on the element to set an optional maximum size.
  • Set data-textfit-adjust="1" on the element to tweak the scaling. 0 = invisible, 1 = normal, 2 = double size
